# About
This repository is Exchange Managment System Web Panel that allows customer to manage their exchange account.🎁


## Features
1. 💸 Manage Remitance
2. 📊 Financial Reports
3. 🧮 Accounting
4. ⚙️ Setting
5. 🔐 Privacy
6. 🀄️ Multi Language(English/Arabic/Farsi)
